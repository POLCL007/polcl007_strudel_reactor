## Introduction
This application was designed for the purpoose of allowing easy editing of strudel music and saving presets

## Working Features

### Json saving and loading
Any strudel song can be saved and loaded as json, complete with the names of instruments, their layers and each modifier.

### Volume adjustment
The volume of the song can be adjusted with a slider

### BPM
The BPM of the song can also be adjusted through a text input
